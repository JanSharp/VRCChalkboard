
# Changelog

## [1.1.0] - 2023-06-11

### Changed

- **Breaking:** Remove and change use of deprecated UdonSharp editor functions (`c258db3`)
- **Breaking:** Use refactored OnBuildUtil (`17dbab8`)

### Added

- Snap RotationGrips before entering play mode or building, using OnBuildUtil (`bb402d6`)
- Add syncing for everyone including late joiners (`a00edb5`, `22c503d`, `b8c9292`, `10e0cea`, `3c8a8a1`)
- Add restriction to one axis (`fecbc90`)
- Support rotating tilted objects (`aedf0b7`)
- Add Snap in line editor feature (`6fa5762`, `3e4a918`)
- Add maximum rotation deviation from original rotation (`5c88825`, `b713043`)
- Use hand bone position in VR (`154f21e`)
- Add snapping of the pickup objects in line with the targeted/rotated object (`a00edb5`, `d5611e2`)
- Add custom inspector (`7f579f1`)
- Add rotation lerping (`ca735f0`)
- Add RotationGrip script (`a00edb5`)
